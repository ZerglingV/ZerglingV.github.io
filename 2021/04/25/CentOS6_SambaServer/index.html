FAQ

Q1. I Can’t Connect The Samba Server By Other CentOS?

when try to connect the server, system shows “Connection to ‘ip’ failed (Error NT_STATUS_HOST_UNREACHABLE)” as figure

solution for Q1

try to allow samba service to be accessible from firewall

Q2. I Can’t Enter My Share Folder On Windows 10?

when try to enter the share folder, system shows “Windows cannot access \\‘your accessing route’. You do not have permission to access \\‘your accessing route’. Contact your network administrator to request access.”

solution for Q2

this is about folder security settings, try to use command as following

#CentOSSamba
